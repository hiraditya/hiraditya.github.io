# About Me

Hi, I'm Aditya Kumar! I am a software engineer specializing in **compilers, static analysis, and software performance optimization**.

## Experience

I have been working as compiler engineer since 2012, where my focus areas include:
- **ML Compilers**: XLA, JAX, MLIR
- **LLVM Compiler Optimizations:** Geared towards improving the runtime performance of native code at massive scale.
- **Clang Static Analysis:** Enhancing and extending static analysis tooling to catch bugs early.
- **Large-Scale Refactoring:** Modernizing codebases and improving architectural efficiency.
- **Performance Analysis:** Identifying and resolving critical performance bottlenecks.

## Projects & Interests

I am passionate about exploring the boundaries of compiler technology, language design, and high-performance computing. You'll often find me experimenting with new optimizations or contributing to systems-level open-source projects.
